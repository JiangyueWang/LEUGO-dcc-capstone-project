#
**LEUGO a one-stop show for your LEGO colleciton**

LEUGO applicaiton is the one-stop shop that designed for the LEGO enthusiasts to view their collected LEGO sets, and the sets in their wish list.

#
**Technologies Used**

Backend: Django REST framework

Database: MySQL

Frontend: ReactJS, CSS

3rd part API and other frontend libraries/packages: Axios, Rebrickable API, Google Chart API, Font Awesome
